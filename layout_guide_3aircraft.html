📐 Blueprint — วิธีจัดวาง 3-Aircraft Petri Net ใน nd.app

แนวคิดหลัก: วาง Shared Resource ให้ตรง x-column เดียวกับ Transition ที่ใช้มัน → เส้นเป็นแนวตั้ง ไม่ทับกัน

🔑 หลักการจัดวาง 5 ข้อ

1
P_Runway → วางด้านบนสุด กึ่งกลาง ระหว่าง T_Land และ T_Takeoff
เส้นจาก T_Land/T_Takeoff ทั้ง 3 ลำจะวิ่งขึ้นตรงๆ แล้วแผ่ออกไปหา Runway
2
P_Bay → วางด้านล่างสุด กึ่งกลาง ระหว่าง T_Dock และ T_Pushback
เส้นจาก T_Dock/T_Pushback วิ่งลงตรงๆ ไม่ตัดกัน
3
P_FuelTruck → วางเหนือ Fueling lane / P_CateringTruck → วางใต้ Catering lane
อยู่ระหว่าง T_StartFuel–T_EndFuel และ T_StartCat–T_EndCat พอดี → เส้นสั้นมาก
4
P_Apron + P_GSEPermit → วางกลางช่องว่างระหว่าง Fuel lane และ Catering lane
ตรง x-column ของ T_StartFuel/T_StartCat → เส้นแนวตั้งสั้น ไม่ต้องวิ่งข้ามไปไกล
5
เว้นช่องว่างระหว่างแถว Aircraft ให้เพียงพอ (อย่างน้อย 150–200px)
เส้น shared arc จะวิ่งผ่านช่องว่างนี้โดยไม่ทับ node ใด
